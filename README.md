## INSTALLATION

### Additional packages

#### Installing yay:
1. ``sudo pacman -S --needed base-devel git``
2. ``git clone https://aur.archlinux.org/yay.git``
3. ``cd yay``
4. ``makepkg -si``


#### Install the following packages using yay:
- visual-studio-code-bin
- pacseek
- wlogout
- waypaper
- blueman-manager
- ckb-next
